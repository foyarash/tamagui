### Usage

```tsx
import { RadioGroup } from 'tamagui'
export default () => (
  <RadioGroup size="$4" value="foo" space="$1">
    <RadioGroup.Item id="dummy" value="foo">
      <RadioGroup.Indicator h="60%" w="60%" backgrounded br={999} />
    </RadioGroup.Item>
    <RadioGroup.Item id="other" value="bar">
      <RadioGroup.Indicator h="60%" w="60%" backgrounded br={999} />
    </RadioGroup.Item>
  </RadioGroup>
)
```

### RadioGroup.Indicator props

RadioGroup.Indicator extends [ThemeableStack](/docs/components/stacks#themeablestack), getting [Tamagui standard props](/docs/intro/props). It appears only when the parent Item is checked.
